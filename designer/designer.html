plotting graphic designer

about

The plotter graphic designer is the software in companion of the plotter arm device.
to show or hide this help message, press the help+/- button on above.

basic usage

You can modify the drawing by dragging the control points on the canvas.
To create a new degree 2 bezier curve, press `new bezier` and click on arbitrary 3 points on the canvas. similarly, you can create straight lines and dots.
if you are satisfied with the graphic, press `export()` to export the graphic data into a format accepted by the plotter arm.
You can also modify the drawing precisely in the below text area. press change() to update the figure.
ctrl+mouse scroll OR ctrl+- to zoom in or out.
if your browser supports local storage, you editing will be cached and restored even after you reload the page. but you should copy and save the data into a .txt file manually for permanent storage.
the undo/redo stack has 12 layers.
you can load the preset figures by pressing `load`

specification of data area

The data area is an array of points.
three points (a,b,c) represent a unit of primitive. if a and b and c are different, then it defines a degree 2 bezier curve, if a equals b but b not equals c, then it defines a straight line. if a equals b equals c, then it defines a single dot.
the last four points (d,e,f,g) are reserved for defining the coordinate space of the exported figure. point e is the relative origin, the distance between (d,e) represents the height of 400 unit which is 4cm. by positioning (d,e,f) carefully, one can scale and position the figure as desired.
normally x value of point g would be zero. if x of g equals 1 then the exported figure will be rotated clockwise 90 degrees.

finally

This software is not a sophisticated graphics designer and is built only by ~500 lines of javascript code. but it is already good enough to design impressive figure/drawing. anyway, the source code of this software is embedded in this HTML file.
